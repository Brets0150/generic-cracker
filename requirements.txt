# MDX Agent - Python Dependencies
#
# This project intentionally has NO external dependencies.
# All functionality is implemented using Python standard library modules.
#
# Python Standard Library modules used:
# - sys: System-specific parameters and functions
# - os: Operating system interfaces
# - argparse: Command-line argument parsing
# - subprocess: Subprocess management (for MDXfind execution)
# - tempfile: Temporary file creation
# - re: Regular expression operations
# - threading: Thread-based parallelism
# - time: Time access and conversions
# - signal: Signal handling (SIGTERM, SIGINT)
# - pathlib: Object-oriented filesystem paths
# - queue: Queue data structure for thread communication
#
# Required Python Version: 3.6+
#
# Installation:
# No pip install needed! Just ensure Python 3.6+ is installed:
#   python3 --version
#
# For development/testing (optional):
# pip install -r requirements-dev.txt  # (if created)
